Tasha's and Josh's Wedding Announcements

[ by Cayleigh Leishman]

Overview

Purpose

To create a website that will allow for people to keep up to date on wedding thingamajigs

Audience

These people are probably going to be their friends and want things that are more personalized to the couple while catering to them. This is mostly an informative website so I want this website to have lots of information. Probably not going to be bombarded with lots of images though.

There will also be some elderly people reading this so I need to make it easy to navigate. The most important thing is that it will be easy to use and read for each user.
